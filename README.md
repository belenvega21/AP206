# Survival Guide - El Reto Móvil

Proyecto desarrollado con Flask para la materia de Programación Móvil.

## Tecnologías utilizadas

- Python
- Flask
- HTML
- CSS
- JavaScript

## Funciones

- Sistema de navegación
- Secciones desbloqueables
- Preguntas interactivas
- Checkbox de compromiso
- Diseño tipo aventura

## Ejecutar proyecto

Instalar Flask:

pip install flask

Ejecutar:

python app.py
